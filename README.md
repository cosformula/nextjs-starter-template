# Nextjs Starter Template

## Feature

- Typescript
- Custom koa server
- Custom layout
- Next-image/Next-css
- Docker
- Nodemon with hot reload
- Yarn

## How to use

Clone the example:

```bash
# Clone the project:
git clone git@github.com:cosformula/nextjs-starter-template.git
cd nextjs-starter-templat

# Install it and run:
yarn
yarn dev

# Your website should be available on http://localhost:3000
```
